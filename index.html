Weather Dashboard

This application utilizes the OpenWeatherMap API and multiple endpoints to provide the current conditions (including the UV index), a five-day forecast, and icons indicating the type of weather for city based on a text string. Work Day Scheduler

This application utilizes Moment.js date formatting to color code hours in the current workday based on whether they are past, current, or future. Appointments are entered and submitted for each hour and saved to local storage where they persist when refreshing the page.
